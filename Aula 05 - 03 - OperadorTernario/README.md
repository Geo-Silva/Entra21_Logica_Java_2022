# Aprendendo sobre Operador Ternario

## If compacto

> Recomendado para pequenas decisões onde resulta em atribuições simples
> Quero atribuir valor em uma variável com base em uma condição (se minha condição ? " retorne isso quando
true" : " retorne isso quando false")

````java

> quantidadeLimite = (minhaCondição ? 10 : 3);

````
