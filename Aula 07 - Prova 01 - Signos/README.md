## Esta primeira parte é sobre um programa que fale o Signo do Usuário

- Eu fiz colocando cada mês como uma função
- Dentro dessa função temos um If que pega o valor do dia
- Então ele compara o dia com o valor de dia correspondente ao limite de algum signo (ex: Sagitário 21 Dez)
- Se o dia for menor ou igual ao valor 21 do exemplo, será sagitário
- Caso contrário resultará em Capricórnio
- Ao final de cada opção dentro dos IFs tem uma linha de retorno ao menu principal (main)
- Também coloquei para saber se um dia é válido para aquele mês, caso ele não possua esse dia (ex, setembro só tem 30 dias)
- Caso seja 31, ele retornará ao menu, já que esse dia não existe no mês.
- O dia e mês são declarados no começo, no menu principal, não dentro das funções
- Assim eu usei o Switch para chamar a função correspondente ao mês que o usuário colocou
