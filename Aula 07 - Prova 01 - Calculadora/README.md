## Esta segunda parte é sobre um programa que seja uma calculadora

- Fiz de maneira simples
- Cada operação é uma função
- No começo o usuário fala qual operação ele quer fazer
- Então isso ativa um switch, que vê a opção correspondente, chama a função e resolve o cálculo
- O usuário então só precisa colocar o primeiro e segundo valor.
- Ao final de todas as funções (ou operações se preferir), ele volta para o menu principal para calcular novamente
