# Aprendendo sobre Operadores Lógicos (Exclusivo para Booleans)

## Tipos

> - E, AND, &&  = (Comparação entre booleans) mais restritivos conforme a tabela verdade

| A | B | RESULTADO |
|---|---|-----------|
| T | T |	 T		|
| T | F |	 F		|
| F | T |	 F		|
| F | F |	 F		|

> - OU, OR, ||  = (Comparação entre booleans) mais inclusivo conforme a tabela verdade

| A | B | RESULTADO |
|---|---|-----------|
| T | T |	 T		|
| T | F |	 T		|
| F | T |	 T		|
| F | F |	 F		|
